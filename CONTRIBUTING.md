## Thank you for contributing to our Aether project!

Currently, we have made the form for creating new issues for our project, which includes:
- **3D Design**: Creating new 3D Design for our robot arm
- **Bug Report**: Reporting the current bug that needs to be fixed
- **Functionality Request**: Requesting new functionality for better performance/design
- **Test Report**: Create new test cases that need to be implemented

First, create a new issue for your work and follow this naming convention for your branch: 
```txt
git checkout -b "issue-#/name-of-the-issue"
```

Second, push all your work to your branch and create a pull request. 
Follow the pull request template that we made.

Lastly, one of our members will review your work and choose whether to merge or reject it. 

Thank you!
